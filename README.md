# DSA Practice IDE

A browser-based C# coding environment for DSA interview prep.

## Features

- 111 LeetCode problems across 24 patterns
- Monaco editor with syntax highlighting and build error markers
- Local .NET execution (runs real C# via `dotnet run`)
- Progress saved automatically to `dsa_progress.json`
- Per-problem guides, YouTube playlists, and multiple attempt tabs
- Canvas-based DSA visualizer

## Setup

Requires [Node.js](https://nodejs.org) and [.NET SDK](https://dotnet.microsoft.com/download).

```bash
npm install
```

Then double-click `start_ide.bat` or run:

```bash
node server.js
```

Open [http://localhost:8080/dsa_ide.html](http://localhost:8080/dsa_ide.html).

## Problems covered

Arrays & Hashing · Two Pointers · Sliding Window · Stack · Binary Search · Linked List · and more.
